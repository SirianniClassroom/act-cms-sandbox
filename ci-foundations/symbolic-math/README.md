Symbolic Mathematics in Python with SymPy
=========================================

This module introduces the cyberinfrastructure skills of performing symbolic
mathematics in Python using the SymPy library.

#### Module Contents

This module contains the following lessons:
1. Symbolic Algebra
2. Symbolic Calculus
3. Operators in SymPy

For lesson details, [launch the `module-contents.ipynb` notebook on
ChemCompute.](https://chemcompute.org/jupyterhub_internal/hub/user-redirect/git-pull?repo=https%3A%2F%2Fgithub.com%2FSirianniClassroom%2Fact-cms-sandbox&urlpath=tree%2Fact-cms-sandbox%2Fci-foundations%2Fsymbolic-math%2Fmodule-contents.ipynb&branch=master)
