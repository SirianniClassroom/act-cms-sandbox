Foundational Calculus
=====================

This module introduces the calculus of scalar- and vector-valued functions of
one or more variables, leveraging SymPy to perform these operations in Python.
While some scientific applications of calculus are included for illustrative
purposes, these lessons are meant neither to introduce nor rigorously treat
these scientific concepts.

#### Module Contents

This module contains the following lessons:
1. Limits & Continuity of Scalar Functions
2. Derivatives of Scalar Functions
3. Integrals of Scalar Functions

The following additional lessons are planned:
1. Series Representations of Scalar Functions
2. Calculus of Real Vector-Valued Functions
3. Ordinary Differential Equations
4. Partial Differential Equations
